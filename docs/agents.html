ZK Transparency
Hidden (ZK-proven)
  • Owner identity (who delegated the agent)
  • Spending limits and budget ranges
  • Cumulative spend caps and history
  • Policy parameters and constraints
  • Delegation chain details
Public (on-chain)
  • Agent wallet address
  • Transaction amounts and destinations
  • Deposits, swaps, LP positions
  • Token balances
  • All tx history on BaseScan

The ZK layer proves the agent is authorized and within budget without revealing the policy. Transactions remain public on Base — true tx privacy would require a shielded chain (e.g., Aztec). Selective disclosure lets auditors verify compliance without seeing the full policy.

Unified Pipeline
Yield Agent scans rates → AI Brain recommends action → ZK Agent proves authorization
→ V4 Hook verifies proof on-chain → Swap via Uniswap → Deposit & Earn (Aave / LP)
→ stETH Treasury funds agent budget via Lido staking yield (Ethereum)
